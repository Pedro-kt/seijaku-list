Política de Privacidad

SeijakuList - Tu compañero de anime y manga

Fecha de última actualización: 26 de Noviembre de 2025

SeijakuList ("nosotros", "nuestro" o "la aplicación") respeta la privacidad de nuestros usuarios ("usuario" o "usted"). Esta Política de Privacidad explica cómo recopilamos, usamos, divulgamos y protegemos su información cuando utiliza nuestra aplicación móvil.

Al usar SeijakuList, usted acepta la recopilación y el uso de información de acuerdo con esta política.

1. Información que Recopilamos

1.1 Información de Registro

Cuando crea una cuenta en SeijakuList, recopilamos:

  • Dirección de correo electrónico
  • Nombre de usuario
  • Contraseña (almacenada de forma cifrada)
  • Información de perfil (nombre, foto de perfil, biografía)

1.2 Información de Uso de la Aplicación

  • Lista de animes y mangas que agrega a su colección
  • Estados de visualización (viendo, completado, pendiente, etc.)
  • Episodios vistos y progreso de visualización
  • Calificaciones y reseñas personales
  • Logros desbloqueados
  • Preferencias y configuraciones de la aplicación

1.3 Información Técnica

  • Tipo de dispositivo y modelo
  • Versión del sistema operativo
  • Identificadores únicos de dispositivo
  • Datos de registro de errores y fallos (crashlytics)
  • Dirección IP

1.4 Información de Terceros

Utilizamos la API de Jikan (MyAnimeList) para obtener información sobre animes, mangas, personajes y estudios. Esta información es pública y no incluye datos personales.

2. Cómo Usamos su Información

Utilizamos la información recopilada para:

  • Proporcionar y mantener la funcionalidad de la aplicación
  • Crear y gestionar su cuenta de usuario
  • Sincronizar sus datos entre dispositivos
  • Personalizar su experiencia en la aplicación
  • Mejorar y optimizar la aplicación
  • Detectar, prevenir y solucionar problemas técnicos
  • Enviar notificaciones importantes sobre la aplicación (si aplica)
  • Generar estadísticas de uso anónimas

3. Almacenamiento y Protección de Datos

3.1 Almacenamiento Local

  • Sus datos se almacenan localmente en su dispositivo mediante una base de datos Room (SQLite)
  • Los datos locales están protegidos por las medidas de seguridad del sistema operativo Android

3.2 Almacenamiento en la Nube

  • Utilizamos Firebase (Google Cloud Platform) para almacenar sus datos en la nube
  • Firebase Authentication para la gestión de cuentas
  • Cloud Firestore para almacenar sus listas y progreso
  • Firebase Storage para almacenar imágenes de perfil
  • Todos los datos en Firebase están cifrados en tránsito y en reposo

3.3 Seguridad

Implementamos medidas de seguridad comercialmente razonables para proteger su información, incluyendo:

  • Cifrado de datos en tránsito (HTTPS/SSL)
  • Autenticación segura mediante Firebase
  • Prácticas de codificación segura

Sin embargo, ningún método de transmisión por internet es 100% seguro.

4. Compartir Información

NO vendemos, intercambiamos ni transferimos su información personal a terceros.

Podemos compartir información solo en los siguientes casos:

  • Proveedores de servicios: Firebase/Google Cloud para hosting y autenticación
  • API de terceros: Jikan API para obtener información pública de anime/manga
  • Requisitos legales: Si es requerido por ley o para proteger nuestros derechos
  • Datos anónimos: Podemos compartir estadísticas agregadas y anónimas

5. Servicios de Terceros

5.1 Firebase (Google)

Utilizamos Firebase para autenticación, almacenamiento y análisis. Firebase puede recopilar información según su propia política de privacidad:

https://firebase.google.com/support/privacy

5.2 Jikan API

Utilizamos Jikan API para obtener información pública sobre anime y manga de MyAnimeList:

https://jikan.moe/

6. Retención de Datos

  • Mantenemos su información mientras su cuenta esté activa
  • Si elimina su cuenta, sus datos personales serán eliminados de nuestros servidores
  • Los datos de registro de errores pueden conservarse de forma anónima para mejorar la aplicación
  • Puede solicitar la eliminación de sus datos en cualquier momento

7. Sus Derechos

Usted tiene derecho a:

  • Acceder a sus datos personales
  • Corregir información incorrecta
  • Eliminar su cuenta y datos asociados
  • Exportar sus datos en formato portable
  • Retirar el consentimiento y dejar de usar la aplicación

Para ejercer estos derechos, puede:

  • Usar las opciones dentro de la aplicación (Configuración → Cuenta)
  • Contactarnos directamente (ver sección de Contacto)

8. Privacidad de Menores

SeijakuList no está dirigida a menores de 13 años. No recopilamos intencionalmente información de menores de 13 años. Si descubrimos que un menor de 13 años nos ha proporcionado información personal, la eliminaremos inmediatamente.

9. Cambios a esta Política

Podemos actualizar nuestra Política de Privacidad periódicamente. Le notificaremos cualquier cambio publicando la nueva política en esta página y actualizando la "fecha de última actualización".

Se recomienda revisar esta política periódicamente para estar informado sobre cómo protegemos su información.

10. Cookies y Tecnologías Similares

La aplicación puede utilizar:

  • Almacenamiento local para preferencias del usuario
  • Tokens de sesión para mantener su sesión activa
  • Estas tecnologías no rastrean su actividad fuera de la aplicación

11. Transferencias Internacionales de Datos

Sus datos pueden ser transferidos y almacenados en servidores ubicados fuera de su país, específicamente en los centros de datos de Google Cloud Platform/Firebase. Nos aseguramos de que cualquier transferencia cumpla con las leyes de protección de datos aplicables.

12. Contacto

Si tiene preguntas o inquietudes sobre esta Política de Privacidad, puede contactarnos:

  • Email: contact@seijakulist.com
  • GitHub: https://github.com/Pedro-kt/seijaku-list
  • Dentro de la app: Configuración → Reportar Error

13. Consentimiento

Al usar SeijakuList, usted consiente nuestra Política de Privacidad y acepta sus términos.
